# Socmed Aggregator

List of routes github-api:

| **Route** | **HTTP** | **Description** |
|-----------|----------|-----------------|
| /api/myrepository | GET | Get all my repository |
| /api/repospublic | GET | Get repository public |
| /api/createrepos | POST | Create new repository |


**Usage**
with only npm: 
```
npm install
npm start
```
